# Ejecutar en desarrolloo
1. clonar el repositorio
2. Ejecutar
```
npm install o npm i
```

3. Tener Nest ClI instalado
```
npm i -g @nestjs/cli
```
4. levantar la basse de datos
```
docker-compose up -d
```

5. Caragar la base de datos 

```
http://localhost:3000/api/v2/seed
```

## Stack usado
*MongoDB
*Nest
